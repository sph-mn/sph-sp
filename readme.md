# sph-sp
digital sound processing with scheme, guile, alsa and au format files

# features
* input/output ports for sound data
* 32 bit float samples

# dependencies
* run-time
  * guile >= 2.2
  * alsa
* quick build
  * gcc and shell for the provided compile script
* development build
  * sph-sc

# installation
```
./exe/compile-c
./exe/install
```

# usage
```
(import (sph sp))
```
